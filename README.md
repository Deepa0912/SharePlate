# 🍽️ SharePlate v2.5.0 - Eradicating Hunger through AI

**SharePlate** is a high-impact, AI-powered food rescue platform designed to recover surplus food from weddings, hotels, and restaurants across India and redirect it to verified orphanages and NGOs in real-time.

---

## 🚀 Key Features (Strategic Polish v2.5.0)

### 🤖 AI-Driven Intelligence
- **Food Intelligence**: Automatic food name and category detection using Gemini AI.
- **Priority Engine**: Real-time matching based on food perishability, expiration, and caloric value.
- **Smart Logistics**: Automated route recommendation and NGO matching for rapid redistribution.

### 🌐 Multi-Lingual Humanitarian Grid
- Native support for **English**, **Kannada (ಕನ್ನಡ)**, **Hindi (हिन्दी)**, **Tamil (தமிழ்)**, and **Telugu (తెలుగు)**.
- 100% localized interface including new Dashboard, Settings, and Help modules.

### 💎 Premium User Experience
- **Impact Center**: Advanced analytics visualizing water saved, CO2 offset, and meals rescued.
- **OLED Dark Mode**: Consistent, accessibility-focused design with local persistence.
- **Interactive Chatbot**: Human-centric AI assistant with mission-specific quick actions.
- **Secure Auth**: Robust JWT-based authentication with seamless token management and session resilience.

---

## 🛠️ Technology Stack
- **Frontend**: Vite + React + Tailwind CSS + Framer Motion (Glassmorphism design).
- **Backend**: FastAPI + MongoDB (BSON abstraction) + Google Gemini AI.
- **Cloud/Media**: Cloudinary (Image management) + Axios (Secure API Interceptors).

## 📦 Getting Started

### Prerequisites
- Node.js (v18+)
- Python (v3.10+)
- MongoDB Atlas account
- Gemini API Key & Cloudinary Credentials

### Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Deepa0912/SharePlate.git
   cd SharePlate
   ```

2. **Frontend Setup**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **Backend Setup**:
   ```bash
   cd backend
   pip install -r requirements.txt
   python main.py
   ```
